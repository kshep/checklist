First Public Release of Jigsaw

"Jigsaw lets us demonstrate a lot of the ideas that have been on the drawing board for years..." -- Tim Berners-Lee, W3C Director

Jigsaw is a modern, extensible web server written in Java(tm), and serves as a testbed for HTTP-NG, mobile code, collaboration, and remote administration.
