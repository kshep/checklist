Adding this page to your web browser's hotlist
will make it easy to add pages to your CheckList as you surf the net.

To add a page to your CheckList, enter the URL of the page in the box below and use the Add Link button to submit the form. If the Use default page title button is checked the page will be added using the title CheckList finds when it first checks the page. If you'd like to enter a different title or add a description for the page, you can check the Edit page title and description button or you can edit the link later.
