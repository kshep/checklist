Add a Link
To add a page to your list, you need to have the URL for the page (the URL starts with "http://"). You can either type in the URL or copy it from your web browser, a mail message, or any other document.

Delete
When you delete a page from your list, CheckList will continue to watch the page for changes, but you won't be notified of these changes. If you add the page to your list later, you will be able to see all the changes since you deleted the page if you'd like.

Edit Link
You can edit the title and description of the link on your CheckList.

Upload a Hotlist
You can upload the hotlist from your web browser or any other document containing links to web pages and CheckList will allow you to selectively add the links in the document to your CheckList.
